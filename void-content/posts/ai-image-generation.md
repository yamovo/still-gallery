## 一张图是怎么生成的

当你输入一段文字，AI 生图模型做的事情可以简单理解为：**从噪声中逐渐还原出符合描述的图像**。

这个过程叫扩散（Diffusion）。模型先看到一团随机噪点，然后一步步去噪，直到画面变得清晰。文字提示词的作用是在每一步去噪时，引导模型朝着你描述的方向走。

理解这一点很重要——模型不是在"画"，而是在"还原"。它不会像人一样构思构图，而是从概率分布中采样。

## 提示词的写法

AI 生图的提示词和文本对话不太一样，它更像**摄影指令**：

### 主体描述
```
一个穿黑色卫衣的年轻人站在雨中的东京街头
```

### 风格修饰
```
电影感，浅景深，35mm 镜头，自然光
```

### 氛围关键词
```
赛博朋克，霓虹灯反光，潮湿的地面，夜景
```

把这些组合起来：
```
一个穿黑色卫衣的年轻人站在雨中的东京街头，电影感，浅景深，35mm 镜头，霓虹灯反光，潮湿的地面，赛博朋克氛围
```

## 几个关键技巧

### 1. 负面提示词（Negative Prompt）

告诉模型你**不想要**什么，和正面描述一样重要：

```
负面：模糊，变形，多余的手指，低质量，水印
```

大多数模型支持负面提示词，它能显著提升画面质量。

### 2. 权重控制

很多工具支持给关键词加权重。比如在 Stable Diffusion 里：

```
(sunset:1.3), beach, (golden hour:1.2)
```

括号和数字表示这个词的影响权重更高。

### 3. 参考图 + 文字

图生图（Image-to-Image）是更可控的方式。上传一张参考图，再用文字描述你想要的变化：

```
保持构图不变，把风格改成水彩画
```

### 4. 种子值（Seed）

同样的提示词，不同的种子会生成不同的图。如果你找到了一张满意的图，记住它的种子值，下次可以基于它微调。

## 模型选择

不同的模型擅长不同的东西：

- **Midjourney**：审美在线，适合艺术创作和概念设计，风格偏电影感
- **DALL·E**：理解力强，适合复杂场景描述，和 ChatGPT 配合好用
- **Stable Diffusion**：开源可本地跑，自由度最高，需要一定技术基础
- **Flux**：新一代开源模型，写实能力强，文字渲染准确

没有最好的模型，只有最适合当前任务的模型。

## AI 生图的局限

### 手和文字
虽然在进步，但复杂的手部动作和长文字渲染仍然容易出错。

### 精确控制
如果你想让画面中某个元素出现在特定位置、保持特定比例，纯文字描述很难做到。这时候需要 ControlNet、Inpainting 等辅助工具。

### 一致性
同一个人物在不同图片中保持一致的外观，目前仍然是个挑战。IP-Adapter 和 LoRA 微调可以部分解决。

### 版权和伦理
AI 生图的训练数据涉及大量受版权保护的作品。商用时需要注意模型的许可协议和当地法规。

## 实际应用场景

- **概念设计**：快速出多个方案，和客户沟通方向
- **社交媒体配图**：生成符合品牌调性的视觉内容
- **插画素材**：为文章、PPT 配图
- **灵感探索**：不确定想要什么的时候，用 AI 碰撞想法

## 最后

AI 生图是一个工具，不是魔法。它能加速创作流程，但审美、构图、叙事——这些核心能力还是在人身上。

学会用 AI 生图的最好方式，不是背诵提示词，而是**多看好的作品，培养视觉直觉**。当你知道什么是好画面，你就知道该往提示词里写什么。
