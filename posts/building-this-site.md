## 技术选型

### 为什么不用框架

React、Vue、Next.js 都很优秀，但对于一个个人博客来说，它们带来的复杂度超过了收益。我选择了最朴素的方式：

- **原生 HTML/CSS/JS**：三个 JS 文件，一个 CSS 文件，没有构建工具
- **Node.js 静态服务器**：零依赖，80 行代码搞定
- **anime.js**：唯一引入的第三方库，负责动画

### 设计语言

```css
:root {
  --bg: #080808;
  --gold: #c9a96e;
  --serif: 'Playfair Display';
  --sans: 'Inter';
}
```

暗色背景 + 金色点缀 + 衬线标题 + 无衬线正文，这套组合简洁耐看。

## 核心功能实现

### 粒子背景

Canvas 绘制的粒子系统，鼠标靠近时粒子会被推开。这个效果给整个页面增加了一层呼吸感。

### 水平拖拽滚动

画廊的横向滚动条支持鼠标拖拽，带惯性衰减。实现时用了 `requestAnimationFrame` 做帧同步，外加指数移动平均做速度平滑。

### Lightbox

点击图片时，从鼠标位置缩放展开——这个细节让交互感觉更自然。关闭时反向缩放回点击位置。
